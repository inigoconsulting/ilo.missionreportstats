ilo.missionreportstats Installation
-----------------------------------

To install ilo.missionreportstats using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``ilo.missionreportstats`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        ilo.missionreportstats

* Re-run buildout, e.g. with:

    $ ./bin/buildout
